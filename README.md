# IRIS
🌸 Iris Flower Classification

This project is a basic machine learning model that classifies iris flowers into three species: Setosa, Versicolor, and Virginica.

📌 Overview
Uses the famous Iris dataset
Applies a classification algorithm
Predicts flower type based on features like petal and sepal length/width
🛠️ Technologies Used
Python
Pandas
NumPy
Scikit-learn
Matplotlib (for visualization)
🚀 Steps
Load the dataset
Preprocess the data
Train the model
Evaluate accuracy
Make predictions
📊 Output
Model accuracy score
Predicted flower species
